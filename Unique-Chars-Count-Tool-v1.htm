Unique Character Extraction Tool v1

This tool helps identify the set of unique characters in a given chunk of text. You can also use this tool simply to add or remove new line in text.

  1. Paste the chunk of text in the box on left and Add Line Breaks. This will add line breaks after each characater and show it on the right side box (auto-selected for you to copy).
  2. Copy this text and paste it in an Excel spreadsheet. Excel will paste each characater in a separate row.
  3. In Excel, select the whole column of cells and use Remove Duplicates feature. This will show you only unique characters and remove duplicate characaters.
